# Connect Linear - Guided Setup

Click the **"Start Linear Setup"** button below to begin a step-by-step guided setup!

## What You'll Need

1. **Linear API Token**
   - Get it from: [linear.app/settings/api](https://linear.app/settings/api)
   - Tip: Keep the tab open so you can easily copy it

2. **Organization Slug**
   - This is from your Linear URL
   - Example: If your Linear is `app.linear.app/mycompany`, your slug is `mycompany`

3. **(Optional) Team Selection**
   - Filter tickets by specific team
   - Or see all tickets across teams
