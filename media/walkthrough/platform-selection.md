# Choose Your Platform

DevBuddy is a multi-platform extension that works with various ticket management systems.

## Supported Platforms

### ✅ Linear
**Full Feature Support**
- Complete ticket management
- Branch creation & associations
- AI-powered PR summaries
- AI-powered standups
- TODO to ticket conversion
- Chat participant integration
- Webview panels for detailed views

### ✅ Jira Cloud
**Core Feature Support**
- View and manage issues
- Create & update issues
- Workflow transitions
- JQL search support
- Comment management
- Sprint & board integration
- Runtime validation (Zod v4)

**Coming Soon for Jira:**
- Branch integration
- AI features
- Webview panels
- Custom fields UI

### ⏳ Coming Soon
- **Jira Server/Data Center** - Self-hosted Jira support
- **Monday.com** - Work OS platform
- **ClickUp** - Project management
- **GitHub Issues** - Native GitHub integration
- **GitLab Issues** - Native GitLab integration

## How to Select Your Platform

1. Open Settings: **Cmd/Ctrl + ,**
2. Search for: `devBuddy.provider`
3. Choose your platform:
   - `linear` - For Linear users
   - `jira` - For Jira Cloud users

You can change this setting anytime to switch platforms!

## What's Different Between Platforms?

### Common Features (Work Everywhere)
- ✅ View tickets/issues in sidebar
- ✅ Create new tickets/issues
- ✅ Update status
- ✅ Add comments
- ✅ Search and filter
- ✅ Chat participant (`@devbuddy`)

### Platform-Specific Features
Some features are currently available only on certain platforms, but we're working to bring feature parity across all platforms.

See the **Feature Compatibility Matrix** in the documentation for a complete comparison.

## After Selection

Once you've chosen your platform:

1. **Linear users**: Configure your Linear API key
2. **Jira users**: Run the Jira Cloud setup wizard

The extension will guide you through the setup process!
