## Command Palette

Access DevBuddy features from anywhere:

1. Press **Cmd/Ctrl + Shift + P**
2. Type "DevBuddy"
3. See all available commands

Quick access commands:
- `DevBuddy: Show Help & Tutorials`
- `DevBuddy: Update Linear API Key` (Linear users)
- `DevBuddy: Setup Jira Cloud` (Jira users)
- `DevBuddy: Refresh Tickets`
- `DevBuddy: Create New Ticket`

## Settings & Customization

Make DevBuddy work your way:

### Open Settings
1. Press **Cmd/Ctrl + ,** to open Settings
2. Search for **"DevBuddy"**
3. Customize your preferences

### Key Settings

**Platform Selection**
- `provider` - Choose your platform (linear, jira)

**AI & Summarization**
- `ai.model` - Choose your AI model (auto, GPT-4o, etc.)
- `writingTone` - Set tone for summaries (professional, casual, technical, concise)
- `ai.disabled` - Disable AI features (privacy mode)

**Workflow**
- `baseBranch` - Default branch for comparisons (main, master, develop)
- `branchNamingConvention` - Branch naming style (conventional, simple, custom)
- `customBranchTemplate` - Define your own branch template

**Monorepo**
- `packagesPaths` - Where your packages live (packages/, apps/, etc.)
- `maxPackageScope` - Warning threshold for PR size

**Linear Integration** (when provider = "linear")
- `linearTeamId` - Filter tickets by team
- `autoRefreshInterval` - How often to refresh (minutes)
- `preferDesktopApp` - Open Linear in app vs browser

**Jira Integration** (when provider = "jira")
- `jira.cloud.siteUrl` - Your Jira Cloud site URL
- `jira.cloud.email` - Your Jira account email
- `jira.defaultProject` - Default project key
