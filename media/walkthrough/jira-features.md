# Jira-Specific Features

DevBuddy supports these powerful Jira features:

## Current Features
- **JQL Search** - Use Jira Query Language for advanced filtering
- **Workflow Transitions** - Complete workflow support with all status options
- **Sprints & Boards** - View and manage sprint information
- **Runtime Validation** - Production-grade API validation

- AI-powered features (PR summaries, standup generation)
