# Connect Your Platform

Choose your platform below to start the guided setup.

## Linear

**What you'll need:**
- Linear API Token from [linear.app/settings/api](https://linear.app/settings/api)
- Your organization slug (e.g., `mycompany` from `app.linear.app/mycompany`)

**The guided setup will:**
1. Help you paste and validate your API token
2. Configure your organization
3. (Optional) Select a team to filter tickets

**[Setup Linear](command:devBuddy.walkthroughSetupLinear)**

---

## Jira Cloud

**What you'll need:**
- Any URL from your Jira (we'll extract the site automatically!)
- Your Jira email address
- API token from [id.atlassian.com](https://id.atlassian.com/manage-profile/security/api-tokens)

**The guided setup will:**
1. Detect your Jira site from any URL you paste
2. Collect your email
3. Help you create and save an API token (we'll open the page for you!)

**[Setup Jira Cloud](command:devBuddy.walkthroughSetupJira)**
